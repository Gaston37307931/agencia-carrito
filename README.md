# AGENCIA IA - SISTEMA LOCALES CON PEDIDOS

Este sistema está diseñado para vender apps simples a negocios que reciben pedidos (carnicerías, verdulerías, ropa, etc.)

## Flujo:
1. Captar cliente
2. Conversar
3. Cerrar venta
4. Crear app
5. Hacer seguimiento

---

## Servicios:
- App tipo catálogo + pedido por WhatsApp
- Automatización simple
- Mejora de ventas

---

## Objetivo:
Generar ingresos rápidos y escalables.
